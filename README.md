# VP CRM (Frontend)

Frontend do CRM da Vertical Partners em Next.js (App Router) + TypeScript + Tailwind + shadcn/ui.

## Rodar localmente

```bash
npm run dev
```

Acesse: http://localhost:3000

## Rotas desta etapa

- `/app/painel`

Estados do painel (via query param):

- `/app/painel?estado=loading`
- `/app/painel?estado=empty`
- `/app/painel?estado=error`
- `/app/painel?estado=denied`
